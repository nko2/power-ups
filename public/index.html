Who we are

We are a group of people who has an arduino lover member. So we wanted to put toghether node.js and ardunio some how. Our idea was to develop a battleship like game in three dimension, played in a cube of leds.

The hardware

To build the game space we built a cube of leds, controlled by an arduino micro-controller, like the one in te picture below.

The player is able to point a point int the 3 dimensional space by using three knobs, one for each axis. And there is a fire button for the player to make an atempt to shop an enimies "ship".

Infortunatly we didnt manage to get a working version in time. We were able to put the hardware together, as you can se from the video above, and the server side is working as far as we could test :). The game interface is still missing, so its not playable yet.

The architeture

The idea was to have a led cube communicating with a tiny server running on a machine. This server would receive all the input comming from the led cube and forward then to the game server. The game server would display the shots you have already attempted, the game score

Our code

In our git repository we have all the code we have up to the moment, including this website and the code put into the arduino. In the branch master is the code of the game server (in node.js). In the branch arduino is the code that runs in the arduino and the serve who listens to the arduino input and forwards them to the game server. And finaly in the branch website is the code for this web page.
